# Protolink

A framework for building and managing agent-based systems with support for multiple transport protocols and discovery mechanisms.

## Features

- Agent-based architecture
- Multiple transport protocols (JSON-RPC, Runtime)
- Service discovery
- Task management
- Message passing
- Context management

## Installation

```bash
pip install protolink
```

## Quick Start

```python
from protolink.agent import Agent

# Create a new agent
agent = Agent(name="example_agent")

# Start the agent
agent.start()
```
